Key

TLB: Checks if the virtual address is already translated. If hit, skip page table.

Page Table: Translates virtual address to real address on TLB miss.

Cache: Fast memory close to CPU. If miss, go to main memory.

Main Memory: Larger, slower memory. If you want, you can add a "miss" step to go to disk.

Disk: Final fallback for data. Always "hits" eventually if reached.
